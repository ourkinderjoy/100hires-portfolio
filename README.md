# 100Hires Portfolio Project

## Tools yang Diinstall
- Cursor IDE (versi 3.5.38)
- Extension Claude Code for VS Code (by Anthropic)
- Extension Codex - OpenAI's coding agent

## Langkah yang Diselesaikan
1. Download dan install Cursor IDE dari cursor.com
2. Install extension Claude Code di Cursor
3. Install extension Codex di Cursor
4. Membuat akun GitHub
5. Membuat repository publik 100hires-portfolio
6. Membuat file README.md ini

## Kendala dan Solusi
- Tidak familiar dengan Cursor IDE → mencari tutorial dan mengikuti panduan langkah demi langkah
- Belum pernah membuat repository GitHub → mengikuti panduan resmi di github.com
